Ca sa pornesti aplicatia:

- Intai deschizi rabbit la localhost:15672
- Creezi exchange libraryapp.direct
- Creezi queue libraryapp.queue 
- Creezi queue libraryapp.queue1
- Creezi binding uri pe exchange 
  - libraryapp.queue are libraryapp.routingkey
  - libraryapp.queue1 are libraryapp.routingkey1
  - le gasesti in application.properties/ direct in cod in kotlin

- Pornesti aplicatia de la butonul verde din fun main (evident deschizi locatia direct din folderul LibraryApp in kotlin)
- Pornesti aplicatia tot de la main in PyCharm cu locatia in qt_gui
- ca sa modifici ceva in interfata deschizi un terminal si scrii designer
